# 복약 트래커

복용 간격과 하루 최대치를 기준으로 **다음 복용 가능 시각**을 계산해 주는 로컬 전용 PWA.
서버 없이 모든 데이터는 기기의 브라우저(localStorage)에만 저장됩니다.

## 기능

- 약마다 기록 방식 선택: **간격 트래커**(다음 복용 가능 계산) 또는 **복용 체크**(먹었는지만 기록)
- 간격 트래커형: 최소 복용 간격(시간)·1일 최대 개수 등록, 다음 복용 가능까지 **남은 시간을 원형 카운트다운 링**으로 실시간 표시 (원 안 카운트다운, 원 밖 다음 복용 가능 시각)
- 오늘 복용량 / 일일 최대치 배지, 최대치 도달 시 '오늘 최대' 고정·초과 시 경고 배너
- **카드를 누르면 약 상세**: 대형 히어로 링(실시간 카운트다운), 오늘 복용량·1일 최대·최소 간격 요약, 최근 30일 복용 내역(한글 시각, 시:분:초 수정·삭제), **지난 날짜 기록 직접 추가**, 약 정보 수정
- **달력**: 날짜별 복용 기록 확인·시간 수정·삭제
- **생리주기 트래킹** (기본 꺼짐, 설정에서 켬): 달력 상단 버튼 → 회차별 기록 목록(기간·주기)·직접 추가·최근 주기 평균 기반 다음 예정일 표시

### 기본 등록 약 / 자주 쓰는 약 선택
설치하면 아래 약들이 미리 등록돼 있고(수정·삭제 가능), 약 추가 폼에서도 선택하면 간격·최대치·단위까지 자동 입력됩니다. 진통제 용법은 식약처 허가 용법 기준:
- 타이레놀정 500mg — 최소 간격 4시간, 1일 최대 8정
- 타이레놀 8시간 이알 서방정 650mg — 최소 간격 8시간, 1일 최대 6정
- 이지엔6프로 (덱시부프로펜 300mg) — 최소 간격 4시간, 1일 최대 4캡슐
- 부루펜정 400mg (이부프로펜) — 최소 간격 4시간, 1일 최대 3정
- 탁센 (나프록센 250mg) — 최소 간격 6시간, 1일 최대 5캡슐
- 로수바이브정 · 본비바정 150mg(월 1회) · 라바로브정 · 라바로하이정 — 복용 체크형

## 설치 (홈 화면에 추가)
- **iPhone (Safari)**: 주소 열기 → 공유 버튼 → "홈 화면에 추가"
- **Android (Chrome)**: 메뉴(⋮) → "앱 설치" 또는 "홈 화면에 추가"
- 설치하면 주소창 없는 전체화면 앱으로 실행되고 오프라인에서도 동작합니다.

## 기술 구성
- Vanilla JS 단일 페이지, 프레임워크·빌드 도구 없음
- `localStorage` 저장 (서버 전송 없음), 날짜가 바뀌면 "오늘 복용량"만 자동 리셋되고 이력은 보존
- PWA: `manifest.json` + 서비스워커(`sw.js`) — 홈 화면에 추가해 앱처럼 설치, 오프라인 동작
- 모바일 우선 반응형, 다크모드(`prefers-color-scheme`) 대응
- 상대 경로만 사용 → GitHub Pages 프로젝트 경로(`/med-tracker/`)에서 그대로 동작

## GitHub Pages 배포
- **Settings → Pages → Source: Deploy from a branch (`main`, `/root`)** 로 설정되어 있음
- `main` 브랜치에 푸시하면 자동으로 재배포됨
- 배포 주소: https://leedalnim.github.io/med-tracker/
